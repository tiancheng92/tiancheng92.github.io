# requests

## 项目简介
* requests包是使用Go语言开发，模仿python的request包写的用于发起http请求，并返回响应结果的第三方包。
* 项目地址：[tiancheng92/requests](https://github.com/tiancheng92/requests)
<!--more-->

## 使用方法
```shell
go get -u github.com/tiancheng92/requests
```
